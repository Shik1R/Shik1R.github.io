Run

Day2 9min15s Recovery Run 今天跑的挺顺利的，虽然才第二天，但其实过去一年零星间断在跑，有时候在想，想要一个所谓权威的计划，一定程度上也是给自己一个暗示，今天跑这么多就足够了，但这种暗示亦是有效的。

这种想法和这两天在读的村上春树的书中也有提到 「其要领与写作长篇小说一般无二，在似乎可以写下去的地方，果决地停下笔来，这样第二天重新着手时便易于进入状态」 「持之以恒，不乱节奏，对于长期作业实在至为重要」

是的，做一件事情最有效的方法是持之以恒，这是自学生时代就明白的道理，但似乎大部分时间都没有做好，总是想要速胜，速成，最终迅速耗尽热情，没有得到真知，这样的状态现在似乎稍微好了一些。

Work

这两天对工作中的一些事似乎略感焦急，可能是贴近年末，想要在今年内完成一些东西，有了一些想法，安排了下去，希望能够有个较好的推进。
