阅读时长: 2 分钟

Run

这周在家里带娃，间隔了好几天没跑步了，时隔近一年的时间，再一次跑了个5km，考虑到太久没跑了，速度压的很低，总体还算舒适，还是要继续下去，任何事情最怕停下来，只要不停下来，慢一点也没什么，会比抢着做些事快很多。

Work

这周到公司完成了述职，确实感觉今年有即将结束的感觉了，也确实是的，马上最后一个月也过了一半了，在这半个月里，做好收尾，想好规划，为明年做打算吧。

Life

前两天去收房了，等待了一年的时间，激动倒是不至于，只是一件件事情顺利的落地，顺利的启动还是让人安心的，后面装修要多考量下，不过和家人已达成一致尽量简单，毕竟将来究竟在哪，其实还是随着孩子的，暂时没有必要搞的太麻烦。

Game

最近几天在玩 Satisfactory 幸福工厂，这游戏大概2-3年前我就买了，当时还是很早期的early access，买的缘由也很简单，因为戴森球计划玩的差不多了，当时因为是3D又不是那么习惯建造逻辑，玩了一个多小时就放着了。 最近看着已经正式版发布了，玩了2-3个小时，还是觉得略有些冗杂了，不过这游戏如果2-3人一起玩倒是乐趣不小，一个人还是太过繁琐，尤其是纯3D铺设也较为复杂，玩过其他建造很容易觉得这里还是有些浪费时间了，估摸着只能短期闲来无事娱乐下，很难真的玩下去了，除非有朋友一起。
